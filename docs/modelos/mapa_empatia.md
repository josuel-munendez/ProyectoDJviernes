# Mapa de Empatía

## Usuario: Administrador del CRM

### ¿Qué PIENSA y SIENTE?
- "Necesito control total sobre los datos del sistema"
- "La seguridad es lo más importante"
- "Quiero estadísticas claras para tomar decisiones"

### ¿Qué OYE?
- "El sistema debe validar todos los campos"
- "Los roles deben estar bien definidos"
- "Hay que prevenir inyecciones SQL y XSS"

### ¿Qué VE?
- Tablas con muchos datos de clientes
- Menús de navegación con opciones de gestión
- Dashboard con números y estadísticas

### ¿Qué DICE y HACE?
- "Revisa que los datos estén completos"
- Exporta reportes a CSV
- Asigna roles a los usuarios
- Gestiona productos y catálogo

### Esfuerzos (Dolores)
- ✗ Asegurarse de que nadie acceda sin permisos
- ✗ Mantener la integridad de los datos
- ✗ Capacitar a los usuarios en el sistema

### Resultados (Ganancias)
- ✓ Control total del sistema
- ✓ Datos centralizados y seguros
- ✓ Reportes rápidos para la toma de decisiones

---

## Usuario: Cliente/Vendedor del CRM

### ¿Qué PIENSA y SIENTE?
- "Quiero encontrar rápido la información que necesito"
- "Espero que el sistema sea fácil de usar"
- "Me preocupa la privacidad de mis datos"

### ¿Qué OYE?
- "El sistema funciona desde cualquier dispositivo"
- "Los datos están protegidos"
- "Se puede usar desde el celular"

### ¿Qué VE?
- Un formulario de login claro
- Tablas con información de clientes
- Botones para crear, editar y eliminar

### ¿Qué DICE y HACE?
- "Necesito registrar un nuevo cliente"
- "Voy a actualizar los datos de este contacto"
- Busca clientes por nombre o email

### Esfuerzos (Dolores)
- ✗ Aprender a usar el sistema
- ✗ Llenar formularios correctamente
- ✗ Recordar la contraseña

### Resultados (Ganancias)
- ✓ Gestión rápida de clientes
- ✓ Acceso desde cualquier lugar
- ✓ Interfaz clara y responsive
